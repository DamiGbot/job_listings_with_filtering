## Languages and Tools

- React
- Javascript
- Scss
- Html


## Installation

1. Clone the repo

```sh
git clone https://github.com/DamiGbot/Photosnap-mutli-page-website.git
```

2. Install dependencies

```sh
npm install
```

3. Start the project

```sh
npm start
```
